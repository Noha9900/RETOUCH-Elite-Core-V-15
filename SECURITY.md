# Security Policy

## Supported Versions
Currently, only **RETOUCH Elite v15.0** is supported for security updates.

## Reporting a Vulnerability
If you find a security flaw, please do not open a GitHub Issue. Instead:
1. Message the developer directly via the Admin ID provided in the bot.
2. Provide a clear description of the exploit.
